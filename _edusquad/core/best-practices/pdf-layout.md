# PDF Educacional — Diretrizes de Estrutura e Layout

## Estrutura de um PDF Educacional

```
Capa (1 página)
Sumário (1-2 páginas)
Introdução (1-2 páginas)
  - Sobre este material
  - Como usar este material
  - Objetivos de aprendizagem
Módulo/Capítulo 1
  - Título e objetivos do módulo
  - Conteúdo principal
  - Exemplos práticos
  - Atividades / Exercícios
  - Resumo do módulo
  - Questões de revisão
[Módulos 2-N seguem a mesma estrutura]
Glossário
Referências Bibliográficas
Gabarito (se houver exercícios)
```

## Elementos de Página

### Hierarquia Tipográfica
- **H1 (Capítulo):** 24-28pt, negrito, cor primária
- **H2 (Seção):** 18-20pt, negrito
- **H3 (Subseção):** 14-16pt, semi-negrito
- **Corpo:** 11-12pt, regular, espaçamento 1.4-1.6
- **Legendas:** 9-10pt, itálico

### Caixas de Destaque
Use caixas visuais para separar tipos de conteúdo:
- 📌 **Conceito-chave:** definição fundamental
- 💡 **Dica:** sugestão prática
- ⚠️ **Atenção:** erro comum a evitar
- 📝 **Atividade:** tarefa para o aluno
- 🔗 **Saiba mais:** leitura complementar

### Margens e Espaçamento
- Margens: 2.5cm (top/bottom), 3cm (left/right) para impressão
- Espaço entre parágrafos: 6-8pt
- Sangria: 0.5cm ou sem sangria com espaço entre parágrafos

## Regras de Conteúdo

1. **Parágrafos curtos:** máximo 5-6 linhas por parágrafo
2. **Listas:** prefira bullet points a parágrafos longos enumerando itens
3. **Imagens:** toda imagem deve ter legenda e ser referenciada no texto
4. **Tabelas:** título acima, fonte abaixo, alternância de cores para legibilidade
5. **Exercícios:** sempre ao final de cada módulo, antes do resumo
6. **Gabarito:** separado no final, não ao lado das questões
