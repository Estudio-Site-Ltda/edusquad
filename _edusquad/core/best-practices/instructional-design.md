# Design Instrucional — Guia Prático ADDIE

## O Modelo ADDIE

### A — Analysis (Análise)
Antes de criar qualquer conteúdo, analise:
- **Quem são os aprendizes?** Nível, experiência prévia, motivação
- **Qual é o gap de performance?** O que eles NÃO sabem ou NÃO conseguem fazer?
- **Qual é o contexto de aprendizagem?** Online, presencial, mobile, corporativo
- **Quais recursos estão disponíveis?** Tempo, orçamento, ferramentas
- **Quais são as restrições?** Prazo, formato obrigatório, compliance

### D — Design (Design)
Estruture ANTES de escrever:
- Escreva objetivos de aprendizagem com verbos de Bloom
- Defina sequência lógica de conteúdo (do simples ao complexo)
- Escolha estratégias instrucionais adequadas ao público
- Planeje avaliações que meçam os objetivos declarados
- Defina o "esqueleto" do curso antes de preencher o conteúdo

### D — Development (Desenvolvimento)
Produza o conteúdo seguindo o design:
- Escreva conteúdo alinhado aos objetivos (nada extra, nada faltando)
- Use exemplos concretos antes de conceitos abstratos
- Aplique a regra 80/20: 80% exemplos/prática, 20% teoria
- Revise com foco no aprendiz — "o que o aluno fará com isso?"
- Teste com um usuário representativo antes de publicar

### I — Implementation (Implementação)
Entregue o conteúdo:
- Configure plataforma LMS ou canal de entrega
- Faça piloto com grupo pequeno
- Colete feedback inicial
- Ajuste antes do lançamento amplo

### E — Evaluation (Avaliação)
Meça efetividade continuamente:
- **Nível 1 — Reação:** Os aprendizes gostaram? (pesquisa de satisfação)
- **Nível 2 — Aprendizagem:** Aprenderam? (avaliações, pré/pós-teste)
- **Nível 3 — Comportamento:** Aplicaram no trabalho/vida? (acompanhamento)
- **Nível 4 — Resultados:** Gerou impacto? (métricas de negócio)

## Princípios de Mayer (Multimídia)

1. **Coerência:** Menos é mais — elimine elementos decorativos desnecessários
2. **Sinalização:** Destaque o que é importante (negrito, cores, ícones)
3. **Redundância:** Não repita o mesmo conteúdo em texto + narração simultaneamente
4. **Contiguidade espacial:** Coloque texto explicativo próximo à imagem que explica
5. **Contiguidade temporal:** Apresente narração e imagem ao mesmo tempo
6. **Segmentação:** Divida em pequenos pedaços — permita o aprendiz controlar o ritmo
7. **Pré-treinamento:** Apresente vocabulário-chave antes do conteúdo principal
8. **Modalidade:** Use narração + imagem (não texto + imagem) para melhor retenção

## Sequência de Conteúdo Recomendada por Módulo

```
1. Hook / Problema Real (por que isso importa para mim?)
2. Objetivos de Aprendizagem (o que vou aprender?)
3. Conteúdo Principal (teoria mínima necessária)
4. Exemplo Concreto #1 (caso real ou demonstração)
5. Atividade de Prática #1 (o aprendiz tenta fazer)
6. Feedback (o que acertou, o que errou, por quê)
7. Exemplo Concreto #2 (variação ou aprofundamento)
8. Atividade de Prática #2 (desafio um pouco maior)
9. Síntese (recapitulação dos pontos-chave)
10. Avaliação (quiz ou tarefa que mede o objetivo)
11. Próximos Passos (o que estudar a seguir)
```
