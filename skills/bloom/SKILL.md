# Skill: Taxonomia de Bloom

## Diretrizes de Aplicação

Ao produzir qualquer conteúdo educacional com esta skill ativa:

1. **Sempre declare objetivos de aprendizagem** antes de escrever qualquer conteúdo
2. Use a estrutura: "Ao final de {módulo}, o {aprendiz} será capaz de [verbo Bloom] [objeto]"
3. Escolha verbos do nível cognitivo adequado ao escopo:
   - Micro-conteúdo → Níveis 1-2 (Lembrar, Compreender)
   - Módulo padrão → Níveis 2-4 (Compreender, Aplicar, Analisar)
   - Curso completo → Níveis 1-6 com progressão crescente

## Verbos por Nível

| Nível | Verbos |
|-------|--------|
| 1. Lembrar | identificar, listar, nomear, reconhecer, definir, citar |
| 2. Compreender | explicar, resumir, classificar, comparar, interpretar |
| 3. Aplicar | executar, usar, demonstrar, calcular, resolver, implementar |
| 4. Analisar | diferenciar, decompor, examinar, distinguir, questionar |
| 5. Avaliar | julgar, defender, selecionar, avaliar, justificar, criticar |
| 6. Criar | elaborar, projetar, desenvolver, formular, construir, planejar |

## Validação de Alinhamento

Antes de entregar qualquer material, verifique:
- [ ] Cada objetivo tem um verbo de Bloom explícito?
- [ ] As avaliações medem o nível cognitivo declarado nos objetivos?
- [ ] A progressão de complexidade é crescente ao longo do conteúdo?
- [ ] Objetivos de nível 5-6 têm conteúdo suficiente para sustentar o nível?
