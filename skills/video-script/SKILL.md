# Skill: Roteiro de Vídeo Educacional

## Estrutura Obrigatória de Todo Roteiro

1. **Abertura (10-15%):** Hook + Promessa + Preview
2. **Desenvolvimento (75-80%):** Blocos de 3-5 min com conceito → exemplo → aplicação
3. **Fechamento (10-15%):** Síntese + Call to Action + Reforço da promessa

## Formato de Bloco de Roteiro

```
[CENA {N} — {NOME DA CENA}]
TEMPO: {início} - {fim}

VISUAL: {o que aparece na tela}
NARRAÇÃO: {texto exato — coloquial, frases curtas, voz ativa}
B-ROLL: {imagens/clips de apoio sugeridos}
NOTA: {instrução para apresentador ou editor}
```

## Regras de Narração

- Frases com no máximo 20 palavras
- Voz ativa sempre que possível
- Tom conversacional — "você vai ver como...", não "será demonstrado que..."
- 1 minuto de vídeo = 130-150 palavras
- Marque [PAUSA] onde há ênfase dramática

## Checklist de Qualidade do Roteiro

- [ ] Tem hook nos primeiros 30 segundos?
- [ ] A promessa é clara e específica?
- [ ] Cada bloco tem conceito + exemplo + aplicação?
- [ ] A narração é em voz ativa e linguagem acessível?
- [ ] Tem call to action ao final?
- [ ] O tempo estimado é adequado ao escopo?
- [ ] O nível de linguagem é adequado ao público-alvo?
